## 平台简介
一款测试管理平台，框架来源于若依，特别鸣谢若依框架的支持。

若依参考后台模板。
> 如需单应用，请移步 [RuoYi-fast](https://gitee.com/y_project/RuoYi-fast)  `(保持同步更新)`，如需其他版本，请移步 [项目扩展](http://doc.ruoyi.vip/#/standard/xmkz)  `(不定时更新)`
